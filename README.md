# Демонстрация класса Init

Веб-приложение для демонстрации работы класса Init на JavaScript.

## Функциональность

- Генерация тестовых данных
- Фильтрация записей (normal и success)
- Сохранение данных в localStorage
- Визуальное отображение в таблицах

## Запуск

Просто откройте `index.html` в браузере.
